### SKILL: FORMATTING & PLAIN TEXT LAYOUT

- Goal: Create a highly readable, plain-text cover letter optimized for the hh.ru platform.
- Volume limit: Greeting + exactly 4-5 sentences for the main body + Contacts.
- Greeting: Use a universal opening, varying the phrasing (e.g., "Здравствуйте!", "Добрый день!").
- Spacing rules: Strictly use double line breaks (`\n\n`) after the greeting, between text blocks, and before the contacts section. Create visual "air".
- List formatting: hh.ru does not render Markdown. If a list is necessary, simulate it using plain text: start a new line with a hyphen and a space (`- `). Do not use asterisks or numbers.
- Contacts block: Extract Telegram and Email from the provided resume. Format them strictly on a single line at the very bottom: `Telegram: [value] | Email: [value]`.
- Keyword constraint: Use the exact Russian word "рад" strictly one time in the entire text.
